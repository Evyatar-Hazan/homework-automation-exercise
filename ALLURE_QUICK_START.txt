================================================================================
                            TROUBLESHOOTING
================================================================================

Problem: "allure: command not found"
Solution: Install allure-commandline
  $ pip install allure-commandline

Problem: "No such file: allure-results/"
Solution: Run the test first
  $ pytest tests/test_ebay_login_allure.py -v --alluredir=allure-results

Problem: "Browser not launching"
Solution: Install Playwright browsers
  $ playwright install chromium

Problem: Port already in use when serving report
Solution: Use a different port
  $ allure serve allure-results/ --port 8080

Problem: Screenshots not showing in report
Solution: Ensure images are properly attached
  $ ls -la allure-results/*.png
