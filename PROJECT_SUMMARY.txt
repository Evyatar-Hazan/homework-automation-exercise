================================================================================
                           HOW TO USE
================================================================================

1. INSTALL DEPENDENCIES
   pip install -r requirements.txt
   playwright install

2. CREATE PAGE OBJECTS
   - Inherit from BasePage
   - Define UI elements as SmartLocators
   - Implement business methods
   - Example: automation/pages/ebay_example.py

3. WRITE TESTS
   - Create fixtures for setup/teardown
   - Use Page Objects for actions
   - Focus on business logic
   - Example: tests/test_ebay_example.py

4. RUN TESTS
   pytest tests/ -v
   pytest tests/test_name.py::test_case -v -s
   pytest tests/ --log-cli-level=DEBUG

5. REVIEW RESULTS
   - Logs: reports/automation.log
   - Screenshots: reports/screenshots/
   - Traces: reports/traces/
   - Videos: reports/videos/

================================================================================
                         IMPORTANT PRINCIPLES
================================================================================

⚠️ DO NOT:
   - Use Playwright directly in tests or Page Objects
   - Hardcode selectors outside of Page Objects
   - Ignore logging - it's your debugging tool
   - Forget to clean up resources in teardown

✓ DO:
   - Use SmartLocator with at least 1 fallback selector
   - Implement high-level business methods in Page Objects
   - Keep tests focused on business logic
   - Call cleanup methods properly
   - Review logs on failures
